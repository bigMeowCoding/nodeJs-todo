# nodeJs-todo
nodeJs 的 命令行 todo list 工具
## 效果
<img width="450" alt="image" src="https://user-images.githubusercontent.com/24516369/169649478-22abb7a0-a310-41ff-a482-0100e3884983.png">

### 支持功能
* 添加任务
* 删除任务
* 修改任务状态
## 本地开发
```
ts-node cli.ts add "任务名称"
```
## 发布
```
npm run build 
npm publish
```
